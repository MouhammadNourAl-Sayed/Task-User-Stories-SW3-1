# Task-User-Stories-SW3-1
## 📌 Project Description

This project represents a simple task management system similar to Trello. It focuses on defining user stories for key features.

---

## 🚀 Features

### 1. User Authentication

* As a user, I want to create an account in order to I can use the system
* As a user, I want to log in in order toI can access my tasks
* As a user, I want to log out in order toI can protect my account
* As a user, I want to see an error message if login fails in order toI know what went wrong
* As a user, I want to stay logged in in order toI don’t need to log in every time

---

### 2. Task Management

* As a user, I want to create a task in order toI can track my work
* As a user, I want to edit a task in order toI can update details
* As a user, I want to delete a task in order toI can remove unnecessary tasks
* As a user, I want to mark a task as completed in order toI can track progress
* As a user, I want to set a due date in order toI can manage my time

---

### 3. Board Management

* As a user, I want to create a board in order toI can organize tasks
* As a user, I want to rename a board in order toI can keep it relevant
* As a user, I want to delete a board in order toI can remove unused boards
* As a user, I want to add lists to a board in order toI can categorize tasks
* As a user, I want to move tasks between lists in order toI can manage workflow

---
