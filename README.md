# Task Management System - User Stories

## 📌 Project Description

This project represents a simple task management system similar to Trello. It focuses on defining user stories for key features.

---

## 🚀 Features

### 1. User Authentication

* As a user, I want to create an account so that I can use the system
* As a user, I want to log in so that I can access my tasks
* As a user, I want to log out so that I can protect my account
* As a user, I want to see an error message if login fails so that I know what went wrong
* As a user, I want to stay logged in so that I don’t need to log in every time

---

### 2. Task Management

* As a user, I want to create a task so that I can track my work
* As a user, I want to edit a task so that I can update details
* As a user, I want to delete a task so that I can remove unnecessary tasks
* As a user, I want to mark a task as completed so that I can track progress
* As a user, I want to set a due date so that I can manage my time

---

### 3. Board Management

* As a user, I want to create a board so that I can organize tasks
* As a user, I want to rename a board so that I can keep it relevant
* As a user, I want to delete a board so that I can remove unused boards
* As a user, I want to add lists to a board so that I can categorize tasks
* As a user, I want to move tasks between lists so that I can manage workflow

---
